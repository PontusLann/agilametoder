git branch

med git branch kan man se alla lokala branches som existerar i sin git repository

git branch -a

med git branch -a kan man se alla lokala och o-lokala branches i sin git repository

git branch -c

git branch -c kan man använda ifall man skulle vilja kopera en branch

git branch -d

git branch -d används för att delete:a en lokal branch, men kommer inte funka om det finns icke mergat material

git branch -m

detta git kommand används för att byta namn på sin branch, kan användas om det blir konflikter vid merge eller liknande

git branch -r

används för att see alla branches som inte är lokala

git push --delete

detta kommando används ifall man skulle vilja ta bort en icke lokal branch

git push --set-upstream

detta kommando används för att skappa en upstream branch, när dett kommando körs kommer din lokala branch pushas till en ny remote branch

git checkout

detta kommando avänds för att byta från en branch till en annan

git checkout -b

detta kommando används ifall man vill göra en ny branch och direkt byta till den nya branchen

git checkout -b /

detta kommando används för att skapa en lokal branch från en remote branch och sedan byta till den branchen

git checkout
