git cherry-pick [insert commit reference]

Tillämpa en commits ändringar på en annan gren.

git clone

Klona ett specificerat repository

git clone

Klona ett arkiv och namnge den lokala katalogen

git clone --origin

Klona ett arkiv och namnge fjärrkontrollen (). Om du inte vill namnge fjärrkontrollen, kommer Git att tillhandahålla standardnamnets ursprung.

git clone --branch

Klona ett arkiv och kolla in den specifika grenen.

git clone --depth

Klona ett arkiv med ett specificerat antal commits

git clone --no-tags

Klona ett arkiv utan att kopiera repets taggar.

git status

Visa en lista över filer i din staging-katalog med tillhörande filstatus.

git add

Om du kör det här kommandot med ett associerat filnamn kommer filändringarna att placeras i din mellanlagringskatalog.

git commit

Spara ändringar i ditt Git-förråd. Om du kör det här kommandot med ett associerat filnamn sparas filändringarna i din repo.

git commit -a

Lägg till alla ändrade och borttagna filer i din arbetskatalog till den aktuella commit.

git commit --amend

Redigera ett Git commit-meddelande genom att lägga till ett meddelande inom citattecken efter kommandot.

git commit -m

Lägg till ett Git commit-meddelande. Lägg till ditt meddelande inom citattecken efter kommandot.

git config --global

Anpassa konfigurationer som är lagrade i din hemkatalog och kan skriva över Git config-systeminställningar

git config --email

Ställer in e-postmeddelandet som är kopplat till dina Git-commits och andra Git-åtgärder.
